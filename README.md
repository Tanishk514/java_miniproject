The Student Management System is a console-based Java application designed to manage students, courses, and enrollments efficiently. The system allows users to add students, create courses, enroll students in courses, and display stored records.

This project demonstrates the use of Object-Oriented Programming (OOP) principles and Java Collection Framework.

objective ----

To manage student and course records.

To implement student-course enrollment functionality.

To use Java Collections like HashMap and ArrayList.

To practice input validation and menu-driven programming
